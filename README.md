# GiftOps

Простой Flask-приложение, которое отдаёт лендинг корпоративных подарков.

## Запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите сервер:
   ```bash
   python app.py
   ```
3. Откройте [http://localhost:5000](http://localhost:5000) в браузере.
